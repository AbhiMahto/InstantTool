# ✨ InstantTools — Instant AI Passport Photo & Privacy-First PDF Toolkit

A fully client-side toolkit that works 100% offline. Create professional passport photos in seconds and process PDFs entirely on your device using WebAssembly + JavaScript. **No uploads. No servers. Complete privacy.**

---

## 📸 Main Highlight: AI Passport Photo
*   **One-Click Generation**: Create professional passport-size photos in seconds.
*   **AI Background Removal**: Automatically removes and cleans backgrounds.
*   **Smart Cropping**: Intelligent cropping to standard dimensions.
*   **100% Local**: AI processing happens right in your browser.

---

## 🧩 PDF Toolkit
*   **Organize**: Merge • Split • Extract • Remove • Reorder
*   **Optimize**: Compress • Rotate
*   **Edit**: Watermark • Metadata Editor
*   **Convert**: Images → PDF • PDF → Images (JPG/PNG)

---

## 🚀 Key Features
*   **Works Offline (PWA)**: Installable as an app; continues working without an internet connection.
*   **Privacy First**: No file uploads — your sensitive documents never leave your machine.
*   **High Performance**: Powered by WebAssembly + Web Workers for near-native speed.
*   **Modern UI**: Sleek, responsive, and easy-to-use interface.

---

## 🛠 Tech Stack
*   **Frontend**: React • Tailwind CSS • Framer Motion
*   **PDF Processing**: pdf.js • pdf-lib
*   **Logic**: WebAssembly • Web Workers
*   **Standard**: PWA (Progressive Web App)

---

## 🔐 Privacy Policy
InstantTools respects your absolute privacy. 
*   **No Uploads**: Files are processed in memory within your browser.
*   **No Tracking**: No cookies, no analytics, no logs.
*   **No external APIs**: Every operation is performed locally.
