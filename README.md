## 📸 Main Highlight: AI Passport Photo Studio

Turn any photo into a professional passport-sized image perfectly matched to your requirements.

- **⚡ One-Click Generation:** Upload and instantly convert.
- **🪄 Smart AI Background Removal:** Automatically isolates the subject and provides a clean backdrop.
- **✂️ Intelligent Cropping:** Detects faces and crops accurately to standard dimensions.
- **🔒 100% Local:** Your face is processed strictly within your browser. No data leaves your device.

## 💻 Quick Start (Dev)

Want to run this locally? Let's get started.

```bash
# 1. Clone the repository
git clone https://github.com/AbhiMahto/InstantTool.git

# 2. Navigate to the directory
cd InstantTool

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

## 🔐 Strict Privacy Policy

**InstantTools respects your absolute privacy.** 
Unlike other free online tools that use your data for training or ads:
- **No Uploads:** Files are processed completely in memory within your browser environment.
- **No Tracking:** Zero cookies, zero analytics, zero logs.
- **No External Servers:** Every single operation is performed locally on your CPU.
